# UI Window Record Specification

## File
`core/include/ui/ui_window_record.h`

## Purpose
Provides inline query functions for `UI_Window_Record` — a registration entry
that describes how to load and close a UI window of a given
`Graphics_Window_Kind`, along with its resource requirements.
